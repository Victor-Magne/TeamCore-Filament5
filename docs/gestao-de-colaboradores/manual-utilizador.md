# Gestão de Colaboradores - Manual do Utilizador

Este guia explica como gerir o ciclo de vida de um colaborador na Aplicação TeamCore, desde a contratação até à gestão do seu contrato.

## 1. Registar um Novo Funcionário
No menu **Gestão de Funcionários > Funcionários**, pode adicionar novos colaboradores clicando em "Novo Funcionário".

*   **Dados Pessoais:** Preencha o nome, NIF, NSS, data de nascimento e contactos.
*   **Localização:** O sistema permite selecionar o País, Estado e Cidade de forma encadeada.
*   **Unidade e Cargo:** Selecione a que departamento o funcionário pertence e qual o seu cargo.

**Importante:** Ao guardar o funcionário, o sistema faz três coisas automaticamente por si:
1.  Cria uma conta de acesso (Utilizador) para o funcionário.
2.  Gera um contrato inicial com base no cargo selecionado.
3.  Prepara o banco de horas para que ele possa começar a picar o ponto.

## 2. Gestão de Contratos
No menu **Gestão de Funcionários > Contratos**, pode consultar e editar as condições de trabalho.

*   **Condições Salariais:** Pode definir o salário específico de cada colaborador.
*   **Horário de Trabalho:** Defina a hora de entrada prevista e a duração da pausa de almoço. Estes dados são fundamentais para que o sistema detete atrasos automaticamente.
*   **Estado:** Um contrato pode estar Ativo, Terminado ou Suspenso. O sistema apenas processa salários e horas para contratos ativos.

## 3. Descarregar Contrato em PDF
Para cada contrato registado, o sistema oferece uma ação de **"Download PDF"**. Isto gera um documento formatado com os dados do funcionário e as condições contratuais, pronto a ser impresso ou assinado digitalmente.

## 4. O Portal do Funcionário
Cada colaborador tem acesso a uma área reservada (Portal do Funcionário) onde pode:
*   Consultar os seus dados pessoais.
*   Ver o estado do seu contrato.
*   Consultar os seus recibos de vencimento.
*   Aceder a uma cópia do seu contrato em PDF.

## 5. Auditoria de Alterações
Se houver uma alteração num salário ou num dado pessoal, o sistema regista quem fez a alteração e quando. Isto aparece nos **Logs de Atividade** para garantir que todas as mudanças nos dados dos funcionários são transparentes.
