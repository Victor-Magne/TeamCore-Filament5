# Configuração da Empresa - Manual do Utilizador

Este guia explica como configurar a estrutura da sua empresa na Aplicação TeamCore, desde departamentos a cargos e localizações.

## 1. Departamentos e Unidades
No menu **Administração > Unidades**, pode definir como a sua empresa está organizada.

*   **Tipos de Unidade:** Pode criar Direções (nível de topo), Departamentos (nível intermédio) e Secções (nível operacional).
*   **Hierarquia:** Ao criar uma unidade, pode indicar qual é a "Unidade Pai". Por exemplo, o "Departamento de Desenvolvimento" pode pertencer à "Direção Técnica".
*   **Gestores:** Pode atribuir um responsável (Gestor) a cada unidade. Este gestor terá visibilidade sobre os funcionários dessa unidade.

## 2. Cargos e Funções
No menu **Administração > Cargos**, definem-se as profissões existentes na empresa.

*   **Níveis:** Pode distinguir cargos por senioridade (Júnior, Pleno, Sénior, etc.).
*   **Salário Base:** Ao definir um salário base para o cargo, o sistema usará este valor como referência para os novos funcionários contratados para essa função.

## 3. Gestão de Localizações
Para facilitar o preenchimento de moradas e contactos dos funcionários, o sistema inclui uma base de dados de:
*   **Países**
*   **Estados/Distritos**
*   **Cidades**

Estes dados permitem que, ao registar um funcionário, o sistema apresente apenas as cidades correspondentes ao estado e país selecionados, evitando erros de digitação.

## 4. Quem pode alterar estas informações?
Por segurança, apenas utilizadores com o papel de **Administrador** têm permissão para alterar a estrutura organizacional, criar novos cargos ou gerir a base de dados de localizações. Gestores de RH podem consultar estas informações, mas não as podem modificar.
