# Processamento de Salários - Manual do Utilizador

Este guia explica como o sistema calcula os vencimentos e como os gestores de RH podem processar os salários mensais.

## 1. Como o Salário é Calculado
O sistema TeamCore automatiza o cálculo do seu vencimento com base em três fatores:

1.  **Salário Base:** O valor definido no seu contrato de trabalho.
2.  **Horas Extra:** Se tiver um saldo positivo no banco de horas no final do mês, o sistema paga essas horas com um **acréscimo de 50%** sobre o valor da sua hora normal.
3.  **Deduções:** Se tiver um saldo negativo (faltas ou atrasos não justificados), o valor correspondente é descontado do seu salário.

## 2. Processar Salários (Apenas RH)
No final de cada mês, o gestor de RH deve:
1.  Aceder ao menu **Recursos Humanos > Processamento Salarial**.
2.  Clicar no botão **"Processar Salários"**.
3.  Indicar o mês e ano de referência (ex: 2026-04).
4.  O sistema gerará automaticamente os recibos para todos os funcionários ativos.

## 3. Revisão e Aprovação
Após o processamento, os recibos aparecem na lista como "Pendentes". O gestor de RH pode:
*   Abrir cada registo para validar os cálculos.
*   Fazer ajustes manuais se necessário (ex: prémios extraordinários).
*   Alterar o estado para "Processado" ou "Pago".

## 4. Consultar Recibos de Vencimento
Cada funcionário pode aceder ao seu portal e consultar o histórico de pagamentos. No detalhe de cada mês, poderá ver:
*   O valor do salário base.
*   Quantas horas extra foram pagas.
*   Quantos descontos foram aplicados por faltas.
*   O valor líquido final a receber.

## 5. Dúvidas sobre Valores
Caso o funcionário não concorde com os valores apresentados, deve consultar o seu **Banco de Horas** e o **Histórico de Ausências**, pois são esses os dados que alimentam o motor de cálculo salarial.
